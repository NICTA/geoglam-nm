About GEOGLAM Map

GEOGLAM Map is a website for map-based access to information about the state and condition of global rangelands. It provides time-based data on the resources used by livestock, allowing country and regional tracking of livestock production including production of meat, wool, and milk. It has been developed, and is currently hosted, by NICTA for the Group on Earth Observations Global Agricultural Monitoring Rangeland And Pasture Productivity (GEOGLAM RAPP). This website is an extension of the Australian Government's National Map initiative.

GEOGLAM Map is a fully open architecture. When you access data through it, you are typically accessing the data directly from the custodian of that data.

The main datasets provided by GEOGLAM Map are:
  • Rangeland and pasture cover
  • Plant available water content
while ancilliary datasets include:
  • Environmental including rainfall, temperature, and soil moisture
  • Livestock use including biomass use, production, feed efficiency, excretion, and greenhouse gas emissions

To see the data in more detail, please refer to the Data Catalogue in the GEOGLAM Map itself. Click the Data Catalogue tab and expand the Data Set group names.

How to use GEOGLAM Map

Just go to http://map.geo-rapp.org. Most operations are obvious, but you can click the Help and FAQs button near the top of this page for specific instructions.

For further information email geoglam_support@lists.nicta.com.au. Feel free to send us feedback and bug reports.

If you have a dataset you think would suit GEOGLAM Map then please get in contact with us: geoglam_support@lists.nicta.com.au

GEOGLAM Map Software

GEOGLAM Map is Open Source software available as a Github project. The core platform, TerriaJS, was initially developed by NICTA for the NationalMap, but has subsequently been used for other projects. It has been made available to assist with the development of other spatial data viewing applications.
